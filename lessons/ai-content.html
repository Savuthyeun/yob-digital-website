AI Content System — ២.A

Concept

AI Content Creation ត្រឹមត្រូវ = 3 ជំហាន ។ មិនមែន Generate ហើយ Post ភ្លាម — ត្រូវ Plan → Draft → Edit:

1

Plan ជាមួយ AI

ប្រើ Claude plan 30-day content calendar → Topics, Hooks, Format ក្នុង ៣០ នាទី

2

Draft ជាមួយ AI

Prompt → generate ៥ variations → ជ្រើស best ១ → Edit ២០%

3

Edit + Publish

Add personal touch + Khmer local context → Schedule → Post

គន្លឹះ
Rule 80/20: AI Draft ៨០% — អ្នក Edit ២០% ។ កុំ Post ដោយមិន Edit — AI ចេះ Generic, អ្នកចេះ Local context ។

Caption Writing — ២.B

Concept

Caption ល្អ = Hook + Body + CTA ។ ប្រើ Prompt ខាងក្រោម → Claude generate ៥ version → ជ្រើស ១ → Edit ភ្លាម !

ឧទាហរណ៍ — Prompt
CLAUDE PROMPT — CAPTION
Act as a Khmer social media copywriter for business in Cambodia.

Write 5 Facebook captions for:
- Business: [ឈ្មោះ + ប្រភេទ]
- Post goal: [Awareness / Leads / Sales / Engagement]
- Product/offer: [ផលិតផល ឬ Promotion]
- Target audience: [SME / Young adults / Parents / etc.]

Format each caption:
Line 1: Hook (attention-grabbing, question or bold statement)
Line 2–4: Body (benefit-focused, ១–២ pain points addressed)
Line 5: CTA (clear action)
Length: 80–150 words each
Tone: Friendly, casual, mix Khmer + English naturally, include 2–3 emoji
គន្លឹះ
Pro Result: ជំនួស ៣ ម៉ោងនៃការ think + write → ៥ នាទី generate + ១០ នាទី edit = Caption Ready !

Blog & Article Writing — ២.C

Concept

Blog ល្អ = SEO Title + Hook Intro + ៤–៥ H2 sections + CTA ។ AI Generate Structure → អ្នក Add real examples ។

ឧទាហរណ៍ — Prompt
CLAUDE PROMPT — BLOG
Act as a Khmer digital marketing educator.

Write a complete blog post:
- Topic: [ចំណងជើង / Topic]
- Audience: [Creator / SME / Beginner]
- Word count: 600–900 words
- Language: Mix Khmer + English (70% Khmer, 30% English terms)

Structure:
H1: SEO title (include main keyword)
Intro: Hook + Problem (100 words)
H2 sections: 4–5 sections with actionable tips
Each H2: 100–150 words + 1 practical example
Conclusion: Summary + CTA to follow YOB Digital

Include: 1 "YOB Tip" callout box per article
គន្លឹះ
Local Touch: បន្ទាប់ Generate → Add ១ real example ខ្មែរ ក្នុងរៀងរាល់ H2 — reader ជឿជាក់ + Share ច្រើន !

Video Script — ២.D

Concept

Video Script ល្អ = [0:00] HOOK → [Body] Points → [End] CTA ។ Specify platform + length → AI generate timestamp-ready script ។

ឧទាហរណ៍ — Prompt
CLAUDE PROMPT — VIDEO SCRIPT
Act as a Khmer YouTube/TikTok scriptwriter.

Write a video script:
- Video length: [60s / 3 min / 10 min]
- Topic: [Topic]
- Platform: [TikTok / YouTube / Instagram Reels]

Script format:
[0:00–0:05] HOOK — Bold statement ឬ question
[0:05–0:45] CONTENT — Main points (bullet by bullet)
[0:45–0:55] EXAMPLE — Quick real case
[0:55–1:00] CTA — Subscribe / Follow / Comment

Add: [B-ROLL SUGGESTION] as comment under each section
Tone: Energetic, conversational, Khmer youth-friendly
គន្លឹះ
Hook = ១ ភាគ ១០ នៃ Video : ៦០s video → HOOK ត្រូវ capture attention ក្នុង ៥ second ។ ប្រើ Prompt សាកល្បង HOOK ៥ versions → ជ្រើស strongest ១ !

30-Day AI Content System

YOB ប្រើ System នេះ generate content calendar ១ ខែ ក្នុង ២ ម៉ោង :

1

Generate Topics (Week 1)

Prompt: "Give me 30 content topics for [business type] targeting [audience] — mix Education/Entertainment/Promotion 70/20/10"

2

Batch Write (Week 2)

Write ២ weeks of captions at once → Save Google Doc → Edit ១ ម្ដង → Done !

3

Schedule (Week 3)

ប្រើ Meta Business Suite → Schedule posts → Mon/Wed/Fri/Sat = optimal Khmer audience !

4

Analyze + Repeat (Week 4)

Check Reach/Engagement → Create more of top-performing type → Month 2 !
