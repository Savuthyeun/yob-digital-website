Windows Installation

1

Download Font File

Download ZIP → Extract → You'll see .ttf ឬ .otf files

2

Install

Right-click on .ttf file → "Install" ឬ "Install for all users" → Done!

3

Verify

Open Word / Canva → Font list → Search font name → ✅ Appears = installed!

Mac Installation

1

Download + Extract

Download ZIP → Double-click to extract → .ttf / .otf files appear

2

Install via Font Book

Double-click .ttf → Font Book opens → Click "Install Font" → Done!

Web Font — CSS Integration

CSS — Add to your stylesheet
/* Upload font files to your server first */
@font-face {
  font-family: 'YOB-Kantum';
  src: url('fonts/YOB-Kantum.woff2') format('woff2'),
       url('fonts/YOB-Kantum.woff') format('woff');
  font-weight: 400;
  font-display: swap; /* important for performance */
}

/* Use it */
body {
  font-family: 'YOB-Kantum', 'Kantumruy Pro', sans-serif;
}

Canva — Upload Custom Font

Requirement: Canva Pro ឬ Canva for Teams ដើម្បី upload custom fonts ។ Free plan = ប្រើ Google Fonts only
1

Brand Hub

Canva Pro → Brand Hub → Brand Kit → Upload a font

2

Upload .ttf

Select .ttf file → Upload → Font appears in Brand Fonts section

3

Use in Design

Text → Font dropdown → "Brand Fonts" section → Select YOB font ✅

⚠️ Common Issues

  • Font មិនឃើញ ក្នុង Word/Canva: Restart application បន្ទាប់ install · ប្រសិនជានៅ install for current user — try "Install for all users" (right-click → Run as Admin)
  • អក្សរខ្មែរបែក / ចេញ □□□: Font installed ប៉ុន្តែ app ត្រូវ Unicode support ។ ប្រើ Word 2016+ / LibreOffice / Google Docs — ជំនួស Notepad ។
  • Mac: Font Book showing "Duplicate": Font Book → Resolve Duplicates → Keep New ។ ឬ delete old version ចេញពី ~/Library/Fonts/ ហើយ reinstall ។
  • Web font @font-face មិន load: ១) File path incorrect — use DevTools → Network tab · ២) CORS error — host font on same domain · ៣) format() ខុស — check woff2 support
  • Canva: Upload font fails: File size >2MB → compress ។ Canva accepts .ttf, .otf, .woff ។ .woff2 not accepted in Canva !
