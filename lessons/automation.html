ManyChat — Facebook & Instagram DM Bot

Free Plan: Up to 1,000 contacts ។ ល្អប្រើ អ្នក ចាប់ផ្ដើម !
1

Create Account

ចូល manychat.com → Sign up with Facebook → Connect Facebook Page / Instagram Account ។

2

Setup Default Reply

Automation → Default Reply → ដាក់ Message ស្វាគមន៍ + Menu buttons (ឧ. "ចង់ដឹងតម្លៃ", "ចង់ Order", "ទាក់ទងផ្ទាល់") ។

3

Keyword Trigger

Automation → Keywords → ដាក់ keyword: "តម្លៃ", "price", "order" → Auto-reply ជាមួយ info ។ ១ keyword = ១ auto-flow !

4

Collect Lead Info

ក្នុង Flow → User Input → ឱ្យ user ផ្ញើ ឈ្មោះ + Phone → Save ទៅ Custom Field → Export ជា CSV ឬ ភ្ជាប់ Google Sheet ។

5

Comment Auto-Reply

Growth Tools → Comment Growth Tool → Post ណាមួយ → ពេល Comment → Auto-DM user ។ Powerful for giveaway / announcement posts !

Make.com — No-Code Automation Flows

Make (formerly Integromat) = ភ្ជាប់ Apps ផ្សេងៗ ដោយ visual drag-and-drop ។ Free plan: 1,000 operations/month ។

1

Use Case: Facebook Lead → Google Sheet

Facebook Lead Ads → Make.com → Google Sheets ។ ពេល Lead Fill Form → ព័ត៌មានចូលទៅ Sheet ដោយ Auto !

2

Use Case: Telegram Notification

Facebook Comment → Make.com → Telegram Bot Message ។ ពេលគេ Comment Post → អ្នកទទួល Notification Telegram ភ្លាម !

3

Setup Steps

make.com → Create account → New Scenario → Add trigger (Facebook) → Add action (Sheets/Telegram) → Save → Activate ។

Zapier — Quick Automation

Zapier = Easy version of Make ។ Free: 100 tasks/month ។ ល្អសម្រាប់ simple 2-step Zaps:

  • Typeform → Google Sheets — Survey responses auto-saved
  • Gmail → Slack/Telegram — Email notification ភ្លាម
  • Facebook Lead → Gmail — Lead notification email
  • Calendly → Google Calendar — Auto booking sync

Telegram Bot Setup

1

Create Bot

Telegram → Search @BotFather → /newbot → ដាក់ Name + Username → Get Bot Token (save it!)

2

Get Chat ID

Search @userinfobot → /start → Copy Chat ID ។ ប្រើ Group Chat ID ដើម្បី notify ក្រុម!

3

Connect to Make/Zapier

Make.com → Telegram module → Paste Bot Token + Chat ID → Test → ផ្ញើ notification !

⚠️ Common Issues

  • ManyChat មិន Reply: ពិនិត្យ Automation → Default Reply ON/OFF ។ ពិនិត្យថា Facebook Page connected → Settings → Connected Accounts ។
  • Keyword trigger មិនដំណើរការ: Keyword ត្រូវ exact match (case-insensitive) ។ ព្យាយាម clear bot conversation → test fresh ។ ManyChat keyword test: Automation → Keywords → Preview ។
  • Make.com Scenario stopped: ១) Operation limit reached (Free: 1,000/month) · ២) Connection expired — Reconnect app · ៣) Error in module — Check error log → Fix data mapping ។
  • Telegram Bot គ្មាន Notification: ពិនិត្យ Bot Token correct · ពិនិត្យ Chat ID (Group IDs start with -) · Test: send curl request ជាមុន ។
  • Lead មិន Save ទៅ Google Sheet: ពិនិត្យ Sheet permission — Bot account ត្រូវ Editor access · ពិនិត្យ column mapping ក្នុង Make/Zapier ។
