ដំឡើង Khmer Unicode Keyboard នៅ Windows ឬ macOS ក្នុង 5 ជំហានសាមញ្ញ — ហើយ វាយអក្សរខ្មែរបានភ្លាម!

Windows Setup

Windows 10 / 11 — Built-in Khmer Unicode support, no extra software needed!
  1. បើក Settings → Time & Language
  2. ចូល Language & region → ចុច Add a language
  3. ស្វែងរក Khmer → ចុច Next > Install
  4. ចូល Khmer options → បើក Khmer keyboard layout (Unicode)
  5. ប្តូរភាសា: Win + Space ឬ Alt + Shift
ទាញយក Khmer Unicode Pack

macOS Setup

macOS — Built-in Khmer support via System Settings Input Sources.
  1. បើក System Settings (ឬ System Preferences)
  2. ចូល Keyboard > Input Sources
  3. ចុច + → ស្វែងរក Khmer
  4. ជ្រើស Khmer Unicode keyboard → ចុច Add
  5. ប្តូរភាសា: Control + Space ឬពី menu bar

Troubleshooting

  • អក្សរបែក ឬចេញជា ▢: សាកល្បង font Khmer Unicode ដូចជា Kantumruy Pro, Noto Sans Khmer, Battambang
  • មិនឃើញ Khmer keyboard: restart ម្តង ហើយពិនិត្យ Input Source ឡើងវិញ
