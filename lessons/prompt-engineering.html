RTFT Framework — YOB Method

YOB ប្រើ Framework RTFT ដែលចងចាំបានងាយ · Apply គ្រប់ Context:

R

Role — AI ជាអ្នកជំនាញអ្វី?

ប្រាប់ AI ថាត្រូវ play role ណា ។ Role ច្បាស់ = Output professional ជាង ។
"Act as a [expert title] with [X] years in [field]..."

T

Task — AI ត្រូវធ្វើអ្វី?

ប្រាប់ Action ច្បាស់ — Write / Analyze / Summarize / Translate / Compare / Generate ។
"Write 5 ... · Analyze this ... · Summarize into 3 points..."

F

Format — Output ដូចម្ដេច?

ប្រាប់ Structure · Length · Language ។ AI នឹង follow exactly ។
"Return as: numbered list · max 200 words · mix Khmer+English · use headers"

T

Tone — ស្ទីលសរសេរបែបណា?

ប្រាប់ Voice + Register ។ Tone ខុស = output feel wrong ទោះ content ត្រឹមត្រូវ ។
"Friendly · Professional · Casual · Formal · Persuasive · Educational..."

Pro Move: ចាប់ផ្ដើម RTFT ជានិច្ច — បន្ថែម Context (ព័ត៌មាន Background) ក្នុង Task section ដើម្បីទទួលបាន output ដែល specific ជាង generic !

Templates — 💼 ការងារ (Work)

Prompts សម្រាប់ Professional context — Report, Email, Meeting, Analysis :

📧

Professional Email

សរសេរ Email ជំនួស ក្នុង ២ នាទី

EMAIL PROMPT
Act as a professional Khmer business writer.

Task: Write a formal email to:
- Recipient: [Title + Name ឬ Company]
- Purpose: [ចង់ request / confirm / follow-up / decline]
- Key points: [ចំណុច ២-៣ ដែលត្រូវ mention]

Format: Subject line + Body (3 paragraphs) + Professional sign-off
Tone: Polite, direct, formal — mix Khmer + English terms where appropriate
📊

Report / Analysis

វិភាគ Data ឬ Situation ស្ទីល Consultant

ANALYSIS PROMPT
Act as a senior business analyst specializing in Cambodian market.

Task: Analyze the following situation and provide recommendations:
[Paste your data / situation description here]

Context:
- Industry: [ប្រភេទ Business]
- Goal: [ចង់ achieve អ្វី?]
- Constraint: [Budget / Time / Resources]

Format:
- Executive Summary (2 sentences)
- Key Findings (3-5 bullet points)
- Risks to Watch (2-3 points)
- Recommended Actions (numbered, priority order)

Tone: Analytical, objective, actionable — no fluff
📝

Meeting Minutes / Action Items

បំប្លែង notes → structured document

MEETING NOTES PROMPT
Act as a professional meeting secretary.

Task: Convert these raw meeting notes into a structured summary:
[Paste your meeting notes here]

Format:
- Date + Attendees
- Key Decisions Made (bullet points)
- Action Items table: | Who | What | Deadline |
- Next Meeting (if mentioned)

Tone: Neutral, clear, professional · English + Khmer as in original notes

Design Brief Generator

បង្កើត Brief ច្បាស់ ផ្ញើទៅ Designer ឬ AI Image tool

DESIGN BRIEF PROMPT
Act as a senior art director experienced in Cambodian brand design.

Task: Write a detailed design brief for:
- Project: [Logo / Banner / Social Post / Website / Packaging]
- Brand: [Brand name + ប្រភេទ Business]

Design Feedback & Critique

AI Critique design ដូច Senior Designer

DESIGN CRITIQUE PROMPT
Act as a brutal-but-constructive senior UI/UX designer with brand expertise.

Task: Critique this design. [Describe your design OR paste description]

Design type: [Logo / Landing page / Social post / App screen]
Brand goal: [ចង់ communicate អ្វី?]
Target audience: [Who sees this?]

Evaluate:
1. Visual hierarchy — Does eye flow correctly?
2. Color usage — On-brand? Accessible?
3. Typography — Readable? Appropriate?
4. Message clarity — Does it communicate in 3 seconds?
5. Improvement priority list (numbered, most impactful first)

Tone: Honest, direct, actionable — no empty praise

Explain Difficult Concept to Students

AI ធ្វើ Analogy + ការពន្យល់ Level-appropriate

EXPLAIN TO STUDENTS PROMPT
Act as a gifted teacher who makes complex ideas simple.

Task: Explain [Concept] to students who are [Age / Grade / Level].

Context: Students already understand: [Previous knowledge]
Common misconception to address: [What do they usually get wrong?]

Format:
- Start with a relatable analogy from everyday Cambodian life
- Explain in plain language (avoid jargon · define if must use)

- Common mistake + how to avoid

Tone: Warm, encouraging, patient — make students feel "I can do this"

Tone: Fair, objective, clear — students should know exactly how to get full marks

Advanced Techniques

Master ៤ technique ខាងក្រោម — output quality jump ១ level :

Core

Chain of Thought

បន្ថែម "Think step by step before answering" ។ AI ធ្វើ reasoning ច្បាស់ → Fewer errors ។ ល្អសម្រាប់ Math, Logic, Analysis ។

Core

Few-Shot Examples

ផ្ដល់ ២-៣ examples ជាមុន ។ AI copy style + format ។ "Here are 2 examples of what I want: [ex1] [ex2] — Now write..."

Core

Constraint Prompting

ប្រើ "Do NOT..." ឬ "Only use..." ។ AI ជៀសវាង mistake ។ "Do NOT use marketing jargon · Only write in Khmer"

Advanced

Iterative Refinement

AI output = Draft ។ Prompt ម្ដងទៀត: "Make it shorter / more formal / add example" — build on same thread ។

Advanced

System Persona

Set AI identity ជានិច្ច :
"For this entire conversation, you are [Expert]. Never break character."

Advanced

Output Anchoring

ផ្ដល់ partial output ដើម្បី anchor style :
"Start your response with: 'ការវិភាគ...' and follow this structure: [outline]"

Prompt Quality Checklist

មុន Submit Prompt — Check ៧ ចំណុចនេះ :

Role ច្បាស់ — AI ដឹងថាជា expert ផ្នែកណា?
Task = Action verb ច្បាស់ (Write / Analyze / Generate / Summarize)
Context ពេញ — Background info ដែល AI ត្រូវដឹងដើម្បីជួយ
Format specified — Structure, Length, Language ច្បាស់
Tone defined — AI ដឹងថា sound បែបណា?
Constraints listed — "Do NOT..." ឬ "Only..." ដើម្បីជៀស mistake
Output ដំបូង → Iterate — Prompt ជាការសន្ទនា មិនមែន One-shot!
