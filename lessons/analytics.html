KPIs សំខាន់ — Key Metrics to Track

📊

Reach vs Impressions

Reach = Number of UNIQUE people who saw your content ។ Impressions = Total times shown (same person can see multiple times) ។ Track Reach ដើម្បីដឹង audience size ពិត ។

💬

Engagement Rate

Engagement Rate = (Likes + Comments + Shares + Clicks) ÷ Reach × 100% ។ Good: 3–6% ។ Great: 6%+ ។ ប្រសិនជាទាបជាង 1% → Content ត្រូវ review !

🎯

CPC & CPM (Ads Only)

CPC (Cost Per Click) = Budget ÷ Total Clicks ។ Target: $0.05–0.30 for Cambodia ។ CPM (Cost Per 1000 Impressions) = Budget ÷ Impressions × 1000 ។ Target: $0.50–2.00 ។

💰

ROAS — Return on Ad Spend

ROAS = Revenue generated ÷ Ads cost × 100% ។ Break-even: 100% ។ Good: 200–300% ។ Great: 400%+ ។ YOB client average: 380% ROAS ។

📈

Conversion Rate

CVR = Leads ÷ Clicks × 100% ។ Landing page CVR target: 2–5% ។ Messenger CVR target: 15–30% (ក្នុង Cambodia context) ។

ROAS Explained — Cambodia Context

ROAS FORMULA
ROAS = Revenue generated from Ads ÷ Ad Spend

Example:
Ad Spend: $100
Revenue from those Ads: $400
ROAS = $400 ÷ $100 = 4x

Target for Cambodian SME:
- Starter: 2x (break even + some profit)
- Good:    3x
- Excellent: 5x+
Improve ROAS: 1) Better targeting → right audience · 2) Better creative → higher CTR · 3) Better landing page → higher conversion rate

Facebook Page Insights — របៀបអាន

  1. ចូល Facebook Page → ចុច Insights (ឬ Professional Dashboard)
  2. ជ្រើស Date Range: Last 28 days (compare month-over-month)
  3. ពិនិត្យ: Post Reach, Engagement, Top Posts
  4. ស្វែងរក Pattern: Post format ណា (Video/Photo/Text) Perform ល្អបំផុត?
  5. Note: Time of highest engagement → Post ពេលនោះ!
YOB Tip: Export data ជា CSV រៀងរាល់ ១ ខែ → Build Excel/Sheets report → Compare ។ Consistency ក្នុង reporting = ឃើញ trend!

Meta Ads Manager — Reading Results

ចូល: business.facebook.com → Ads Manager → Campaigns tab

Columns ដែលត្រូវ Track:

ADS MANAGER COLUMNS TO ADD
Results          → Lead/Message/Purchase count
Reach            → Unique people reached
Impressions      → Total ad shows
CPR (Cost/Result) → Key efficiency metric
Amount Spent     → Budget used
ROAS             → If selling products (Catalog ads)
CTR (Link)       → Click-through rate on link
CPC (Link)       → Cost per link click

Rule of thumb: Campaign run for 3–5 days minimum before judging performance ។ Facebook Algorithm ត្រូវការ Learning Phase ២–៣ ថ្ងៃ !
