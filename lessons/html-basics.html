HTML + CSS = ភាសា ២ ដែលប្រើ build website ។ HTML = Structure (skeleton), CSS = Design (clothes) ។ រៀន ១ សប្ដាហ៍ = Build Website ផ្ទាល់!

HTML ជាអ្វី?

HTML (HyperText Markup Language) = ភាសា ប្រើ define structure ឬ content នៃ webpage ។

Simple Analogy: HTML = skeleton of body (bones/structure) · CSS = appearance (skin, clothes, style) · JavaScript = movement (muscles, behavior)

Structure មូលដ្ឋាន

HTML — Basic Structure
<!DOCTYPE html>
<html lang="km">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>ឈ្មោះ Page</title>
    <link rel="stylesheet" href="style.css" />
  </head>
  <body>
    <!-- Content goes here -->
    <h1>ស្វាគមន៍!</h1>
    <p>នេះជា paragraph ដំបូង ។</p>
  </body>
</html>

Tags សំខាន់ — ចាំ ១០ Tag នេះ

HTML TAGS REFERENCE
<h1> ដល់ <h6>  — Headings (h1 ធំ, h6 តូច)
<p>           — Paragraph
<a href="">  — Link
<img src=""> — Image
<div>         — Container block
<span>        — Container inline
<ul>/<li>    — Bullet list
<button>      — Button
<form>        — Form container
<input>       — Text input field

CSS Styling — ពណ៌ + Font + Layout

style.css — BASICS
/* Body — overall page */
body {
  background: #080b10;    /* background color */
  color: #ffffff;          /* text color */
  font-family: 'Kantumruy Pro', sans-serif;
  margin: 0;
  padding: 0;
}

/* Heading */
h1 {
  font-size: 48px;
  color: #e52c67;         /* YOB pink */
  text-align: center;
}

/* Button */
.btn {
  background: #e52c67;
  color: white;
  padding: 12px 28px;
  border-radius: 50px;
  border: none;
  cursor: pointer;
}

Your First Webpage — Practice!

1

Open Notepad / VS Code

Windows: Notepad ++ ឬ install VS Code (free) · Mac: TextEdit (set Plain Text)

2

Paste HTML Template

Copy structure ខាងលើ → Paste → Change Title + Content ជា Khmer → Save as "index.html"

3

Open in Browser

Double-click index.html → Opens in Chrome/Firefox → ឃើញ webpage !

4

Add CSS

Create "style.css" same folder → Paste CSS → Refresh browser → ឃើញ styled !
