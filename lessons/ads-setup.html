Reading Results — Key Metrics

  • CTR (Click Through Rate) — Good: >1.5% · Excellent: >3% · Low = change creative
  • CPL (Cost per Lead) — Khmer learners avg: $1–5 · Too high = change audience ឬ creative
  • ROAS — Revenue ÷ Ad Spend · Good: >2x · Excellent: >4x
  • Frequency — How many times same person sees ad · >3 = refresh creative
YOB Rule: Run Ads ៧ ថ្ងៃ before making changes ។ Algorithm ត្រូវ learning phase ។ Change too fast = waste budget!

⚠️ Common Issues

  • Ad rejected / ច្រានចោល: ពិនិត្យ Policy violations — ពាក្យ "FREE", "Guaranteed", before/after images, health claims ។ Edit copy → Resubmit ។
  • Account disabled: ទៅ facebook.com/help/contact → "My ad account was disabled" → Submit appeal ។ ធ្វើ Appeal ១ ដង — មិនត្រូវវាយ Submit ច្រើនដង !
  • ចំណាយ budget ហើយ គ្មាន Lead: ១) Landing page ខ្សោយ — ពិនិត្យ CTA · ២) Audience ទូលំទូលាយ — Narrow targeting · ៣) Creative ពុំទាក់ — A/B test visual ថ្មី
  • Pixel not firing: Events Manager → Test Events → ពិនិត្យ installation · ប្រើ Meta Pixel Helper Chrome extension
  • Learning phase stuck: Campaign ត្រូវការ ≥50 conversions/week ដើម្បីចេញ learning phase ។ ប្រើ Conversions objective ជំនួស Traffic ។
