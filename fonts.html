    ប្រើ Font Khmer ក្នុង Microsoft Word ដូចម្តេច?
    បន្ទាប់ពី install font ហើយ បិទ-បើក Word ម្តង, ទៅ Home > Font list រួចជ្រើសឈ្មោះ font Khmer ដែលអ្នកបានដំឡើង។ ប្រសិនបើមើលមិនឃើញ សាក restart Windows ម្តង។
    ប្រើក្នុង Figma និង Canva ត្រូវធ្វើអ្វីខ្លះ?
    Figma Desktop នឹងអាន font ពីម៉ាស៊ីនដោយស្វ័យប្រវត្តិ (restart app ក្រោយ install)។ សម្រាប់ Canva បើ font មិនមានក្នុង library សាធារណៈ សូម upload font នៅ Brand Kit (គម្រោងដែលគាំទ្រ custom fonts)។
    Photoshop/Illustrator អក្សរខ្មែរបែក ឬជាន់គ្នា តើដោះស្រាយដូចម្តេច?
    ត្រូវប្រើ Khmer Unicode font ពិត និងកំណត់ text engine ឲ្យ supporting complex scripts។ បើនៅតែបែក សូម update Adobe app និងពិនិត្យថា keyboard input ជា Khmer Unicode មែនទេ។
